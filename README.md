### how to use
```
python3 computor.py <polynomial>
```
optional arguments:

`-h` show help message and exit

`-p` <int> set the accuracy of the solution

`-f` show irreducible fraction

`-v` show detail operations

to run random test:

```
python3 test_comp.py
```
